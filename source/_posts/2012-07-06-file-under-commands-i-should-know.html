--- layout: post title: File under 'Commands I should Know' tags: - Commandline - LINUX - Technology status: publish type: post published: true meta: _edit_last: '1' dsq_thread_id: '1045298991' :alias: /2012/07/file-under-commands-i-should-know/ --- [cc_bash]#shutdown now -h[/cc_bash] Pretty important but didn't know about the '-h' flag until today. When run as root this will shutdown system (Linux specific) now and send it to a init-level 0, ie a full halt. Good to know =D. As pointed out in comments 'shutdown -p now' is the BSD equivalent.
