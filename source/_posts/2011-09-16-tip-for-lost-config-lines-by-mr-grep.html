--- layout: post title: Tip for lost config lines by Mr. Grep tags: - BASH - Grep - LINUX - Technology status: publish type: post published: true meta: _edit_last: '1' dsq_thread_id: '1078189926' :alias: /2011/09/tip-for-lost-config-lines-by-mr-grep/ --- When editing multiple config files in complex installs it's easy to lose track of where each parameter exists. Solution? Recursive grep with -n function ie [cce_bash]grep -rn help_me_param . [/cce_bash] -n = line number and file name printed on left column of results -r = recursive -rn = win
